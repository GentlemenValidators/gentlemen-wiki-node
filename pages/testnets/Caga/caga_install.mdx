## Introduction

This project aims to provide installation, running, and maintenance capabilities of CAGA validator node for potential and existing CAGA Blockchain backers.

## Prerequisites

Operating System: Ubuntu >= 20.04 LTS

RAM: 8GB minimum, 32GB recommended

Persistent Storage: 25GB minimum, 100GB high-speed SSD recommended


### Step 1: Update and Upgrade
```bash copy
apt update && apt upgrade -y
apt -y install build-essential tree
```
### Step 2: Clone git repository
```bash sudo
  git clone https://github.com/ethereum/go-ethereum
  cd go-ethereum
```

### Step 3: Install Golang
```bash copy
wget https://go.dev/dl/go1.22.3.linux-amd64.tar.gz
rm -rf /usr/local/go && tar -C /usr/local -xzf go1.22.3.linux-amd64.tar.gz

echo 'export GOPATH=/usr/local/go' >> $HOME/.bashrc
echo 'export GOBIN=$GOPATH/bin' >> $HOME/.bashrc
echo 'export PATH=$PATH:$GOBIN' >> $HOME/.bashrc
source $HOME/.bashrc
```

### Step 4: Build Geth

```bash copy
  make all
```

### Step 3: Download Genesis file and install
```bash copy
wget https://gentlemenvalidators.com/caga
mv caga genesis.json
geth --datadir ./chaindata/node1/ init ./genesis.json
```

### Step 5: Install Config P2P

```bash copy
cd ./chaindata
nano config.toml
```
